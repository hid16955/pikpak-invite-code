## PikPak 초대 코드(Invite Code)란 무엇인가

픽팍에는 **초대(리퍼럴) 프로그램**이 있다. 기존 사용자가 자신의 초대 코드를 공유하고, 새 사용자가 그 코드를 입력하면—양쪽 모두 무료 프리미엄 기간을 받는 구조다.

초대 코드를 통한 가입 방식은 세 가지다:

1. **초대 링크로 설치**: 초대 링크를 통해 앱을 설치하면 자동으로 초대 관계가 연결된다.
2. **파일 공유로 초대**: 공유된 파일을 통해 앱에 접근하는 방법.
3. **초대 코드 직접 입력**: 앱 설치 후 12시간 이내에 초대 코드를 수동 입력하면 초대 관계가 성립된다.

중요한 점은 **등록 후 12시간 이내**에 코드를 입력해야 한다는 것. 이 시간을 넘기면 코드를 입력해도 혜택이 적용되지 않으니 가입 직후 바로 처리하는 게 좋다.

1. 크롬 또는 엣지 브라우저에 픽팍 브라우저 확장 프로그램 설치
2. 확장 프로그램에서 로그인
3. 오른쪽 상단의 "출석 체크하여 무료 프리미엄 받기" 버튼 클릭
4. 3일 연속 출석 후 랜덤 프리미엄 획득

꾸준히 하면 비용 없이 프리미엄을 계속 유지하는 분들도 있다. 단, 운에 따라 받는 기간이 다르다.

- **클라우드 다운로드 속도**: 파일의 80% 이상이 수초 내에 클라우드에 저장된다. 개인 인터넷 속도와 무관.
- **4K 스트리밍**: 저장된 영상을 앱이나 웹에서 바로 4K로 재생 가능.
- **텔레그램 봇 연동**: @PikPak_Bot에 링크나 파일을 전달하면 자동으로 클라우드에 저장.
- **클라우드 압축 해제**: zip, rar, 7z 파일을 다운로드 없이 클라우드에서 바로 풀기 가능.
- **브라우저 확장 프로그램**: 웹 페이지의 마그넷/토렌트 링크를 자동 감지해 원클릭 저장.
- **ISP 감시 차단**: 모든 서버 통신이 암호화되어 ISP에서 저장 파일 내용을 확인 불가.

**아쉬운 점:**

- **업로드 속도**: 불안정하고 대체로 느리다는 사용자 후기가 많다.
- **iOS 오프라인 다운로드 불가**: 앱스토어 정책으로 인해 iOS 앱에서는 오프라인 다운로드가 지원되지 않는다. 웹 브라우저나 텔레그램 봇 경유로는 가능.
- **macOS 클라이언트 미출시**: 현재 개발 중이라고 공지만 되어 있는 상태.
- **무료 계정 저장 용량**: 6GB로 타 클라우드 서비스 대비 적은 편.

## 자주 묻는 질문

**Q. 초대 코드와 초대 링크, 뭐가 다른가?**

초대 링크는 링크를 통해 앱을 설치하는 순간 자동으로 연결된다. 초대 코드는 먼저 앱을 설치하고 로그인한 뒤, 12시간 이내에 직접 입력해야 한다. 이미 앱을 설치했다면 코드 방식을 쓰면 된다.

**Q. 초대 코드로 받는 프리미엄은 글로벌인가 지역인가?**

모두 글로벌 프리미엄으로 제공된다.

**Q. 프리미엄을 해지하면 파일이 사라지나?**

프리미엄 만료 즉시 파일이 사라지지는 않는다. 단, 6GB 초과 용량에 대해서는 스트리밍이 제한되고 다운로드 속도가 100KB/s 수준으로 낮아진다.

**Q. 앱 외부에서 구매한 프리미엄 코드는 믿을 만한가?**

픽팍은 공식 기프트카드 서비스 센터를 운영하며, 프리미엄 코드를 통한 판매는 공식적으로 허용된 방식이다. 단, 코드 이외의 방법으로 판매하는 계정을 넘기는 방식은 불법이다.

**Q. 한국에서 지역 프리미엄을 살 수 있나?**

픽팍이 사용자의 IP, 기기 언어, SIM 카드 국가 코드를 종합해 자동으로 적합한 플랜을 제공한다. 한국 사용자에게는 기본적으로 글로벌 프리미엄이 제시된다.
